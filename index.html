This app is designed for small groups of users with the need to communicate with each other in absolute secrecy.

  •  Without any server in between,
  •  Both orally and with short messages,
  •  Directly, one-to-one,
  •  Being anywhere in the world,
  •  Outside of all known communication channels (Skype, Viber etc),
  •  Avoiding any call registration.

 The logic behind is quite simple: A common email account is used where each user’s device sends its public Internet address. All devices of the group having access to this mailbox can connect to each other with the use of a custom proprietary communication protocol.

 An email account can easily be created in Gmail, Yahoo etc. specifically for this application. Its login details are entered when the app runs for the first time and are stored in each user’s device only. No information about the users’ activity is leaked anywhere else.

 This type of direct connection makes encryption unnecessary.

 The application runs both in iOS and Android devices.
